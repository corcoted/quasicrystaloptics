# quasicrystaloptics
Quasicrystal optical system simulations for the paper found in

Corcovilos, T. A., & Mittal, J. (2019). "Two-dimensional optical quasicrystal potentials for ultracold atom experiments." _Applied Optics_, *58* (9), 2256â€“2263. [doi:10.1364/AO.58.002256](http://doi.org/10.1364/AO.58.002256) or [arXiv:1903.06610](http://arxiv.org/abs/1903.06610).

The code is written in Python 3 as a Jupyter notebook.  It uses the [poppy](https://github.com/spacetelescope/poppy) library to perform Fourier wavefront propagation.
